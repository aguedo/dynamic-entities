
# Dynamic Entities API Roadmap

This roadmap outlines the steps to evolve the current API to match the capabilities of the Relativity Objects API.

## 1. Core CRUD for Entity Types _(High Priority)_
[x] List all Entity Types
[x] Get Entity Type by ID
[x] Update Entity Type
[x] Delete Entity Type



## 2. Dynamic Field Management _(High Priority)_
- [ ] Add Field to Entity Type
- [ ] Remove Field from Entity Type
- [ ] Update Field on Entity Type
- [ ] Support for additional field types (Choice, Date, Boolean, Object Reference, etc.)



## 3. Object Instance Management _(High Priority)_
- [ ] Create Instance of Entity Type
- [ ] Get Instance(s) of Entity Type
- [ ] Update Instance of Entity Type
- [ ] Delete Instance of Entity Type
